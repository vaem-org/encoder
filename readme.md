# VAEM - Encoder

A tool for encoding and managing video assets 

## Getting Started

Start development server using Docker compose
```bash
docker-compose up -d
```

Or using local development (ffmpeg is needed):
```bash
yarn
yarn dev
```

## Built With

* [FFMPEG](http://ffmpeg.org/) - Used for video conversion
